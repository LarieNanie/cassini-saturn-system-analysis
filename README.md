# Cassini Saturn System Analysis

## Overview

This project explores publicly available datasets from NASA's Cassini mission to Saturn.

The objective is to apply data analysis and visualization techniques to spacecraft and orbital datasets in order to better understand mission operations, orbital dynamics, and scientific observations collected throughout the mission.

## Objectives

- Acquire and process Cassini mission datasets
- Explore spacecraft trajectory and orbital information
- Visualize key mission parameters
- Investigate patterns and anomalies in spacecraft measurements
- Develop reproducible scientific analysis workflows
